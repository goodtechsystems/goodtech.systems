How to export as og-image.png:
1. Open this file in Chrome at 100% zoom.
2. Open DevTools (Cmd+Option+I), click the element picker, then click the card below to select #card.
3. In the Elements panel, right-click <div id="card"> → Capture node screenshot.
4. Save the downloaded file as og-image.png in the project root, then close this file.
